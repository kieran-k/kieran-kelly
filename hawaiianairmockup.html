Hawaiian Airlines Mockup

A personal project recreating a simplified mockup of Hawaiian Airlines home page website. I have always admired and enjoyed using Hawaiian Airlines website when planning trips and wanted to create a similar, simplified version to practice my UI skill. This recreation is only for UI, buttons and links do not currently link to anything.

This mockup is built with Javascript, React and Semantic UI.
