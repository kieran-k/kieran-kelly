Twitter Bot

My first personal project, I decided to create a Twitter bot for my own personal use. This project was my first experience using python.

This twitter bot is able to like, follow and retweet. Main use of this bot is to pull information from the users main account. The bot will pull the “Likes” of a user and retweet them as well as like posts containing keywords or hashtags set in the bots code. Purpose for this is to have a separate account to show interests that might not suit main account tweets and serve as a separate reference/blog from main Twitter account.

This bot is written in the python programming language and uses the python library Tweepy to access the Twitter API, tweepy wraps the Twitter API for use with python.
